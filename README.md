# FISIOSELF App Notas VX

Aplicacion clinica privada para notas, valoraciones y expediente del paciente.

Este repositorio es independiente de la web publica.

Stack inicial:

- React + Vite
- Supabase Auth
- Supabase Postgres
- Dictado por voz
- IA clinica mediante proxy seguro

Reglas del proyecto:

- No usar Notion.
- No mezclar con la web publica.
- No importar codigo desde repos viejos sin revisarlo.
- Todo lo relacionado con notas, valoraciones, expediente, IA y Supabase vive aqui.
